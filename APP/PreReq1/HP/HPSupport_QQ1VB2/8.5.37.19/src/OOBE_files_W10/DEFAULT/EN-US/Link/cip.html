Data Collected by HP

If you select to share anonymous usage data with HP, HP Support Assistant will collect and send to HP data related to the usage of your computer. HP does not collect any information on a user’s individual data files. All collected data is anonymous (this data cannot be traced back to you) and is treated according to the HP Privacy Statement, which can be found at www.hp.com/go/privacy .

The usage data helps HP select the software that should be included on future PCs from HP and improve HP applications. The collected data includes, but is not limited to:

  • Use of HP Support Assistant features such as search, tune-up, diagnostics tools, support content, and contact options.
  • Overall usage and performance data of HP applications (for example: which web pages were viewed, how long the HP applications were used on each occasion).
  • Information on non-HP applications such as software application title, version number, usage statistics, and general system information. HP does not collect information on specific actions taken in these applications.

HP Active Health collects the following types of data:

  • Hardware configuration and hardware errors.
  • System configuration and system errors.
  • Application configuration and application errors.
  • Hardware, OS, and software performance attributes.
  • BIOS configuration and BIOS errors.
